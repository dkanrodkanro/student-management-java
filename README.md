# Student Management Project (Java Console)

## 소개
JAVA로 만든 콘솔 기반 학생 관리 프로그램입니다. 학생 정보를 등록, 조회, 검색, 수정, 삭제할 수 있습니다.

## 기술 스택
- JAVA
- IntelliJ IDE
- Git/GitHub

## 주요 기능
- 학생 등록
- 전체 조회
- 이름 검색
- 정보 수정
- 정보 삭제

## 실행 방법
1. 프로젝트 클론
2. Intellij나 eclipse에서 실행
3. StudentApp 실행
